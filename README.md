# 컴퓨터 구조 



```
컴퓨터 구조 스터디를 위한 저장소입니다.
각자 공부한 것을 정리해서 올리는 저장소입니다.

교재 : 밑바닥부터 만드는 컴퓨팅 시스템 - 불 논리부터 컴퓨터 아키텍처, 운영체제까지
```

### 스터디 규칙

 	1. 매주 화요일 20:00에 스터디 진행
 	2. 추가 예정
